Reference Formulas

Chi-squared test statistic: \[\chi^2 = \sum \frac{(O - E)^2}{E}\]

Expected count (goodness-of-fit, equal proportions): \(E = n/k\) where \(k\) = number of categories.

Expected count (test of independence): \[E = \frac{(\text{row total})(\text{column total})}{\text{grand total}}\]

Degrees of freedom: goodness-of-fit: \(df = k - 1\)  ·  independence: \(df = (r-1)(c-1)\)
