Math 146 — Week 7 Prequiz Practice

Two-sample inference for means and proportions. Expand each solution to check your work.

Reference Formulas

Standard error for two independent means: \[ \text{SE}_{\bar{x}_1 - \bar{x}_2} = \sqrt{\frac{\sigma_1^2}{n_1} + \frac{\sigma_2^2}{n_2}} \]

Test statistic and confidence interval for \(\mu_1 - \mu_2\): \[ z = \frac{(\bar{x}_1 - \bar{x}_2) - 0}{\sqrt{\dfrac{\sigma_1^2}{n_1} + \dfrac{\sigma_2^2}{n_2}}} \qquad\qquad (\bar{x}_1 - \bar{x}_2) \pm z^* \sqrt{\frac{\sigma_1^2}{n_1} + \frac{\sigma_2^2}{n_2}} \]

Pooled proportion and test statistic for \(p_1 - p_2\): \[ \hat{p}_{\text{pool}} = \frac{x_1 + x_2}{n_1 + n_2} \qquad\qquad z = \frac{\hat{p}_1 - \hat{p}_2}{\sqrt{\hat{p}_{\text{pool}}(1-\hat{p}_{\text{pool}})\!\left(\dfrac{1}{n_1}+\dfrac{1}{n_2}\right)}} \]

Problem 1 — Independent or Paired?

For each scenario, say whether the data are independent or paired.

  1. The same 30 students take a test before and after a review session.
  2. Two different sections of a class use different textbooks and are compared.
  3. The blood pressure of each patient is measured before and after taking a medication.
  4. Support for a ballot measure is surveyed in two different cities.
Show Solution
(a) Paired
The same 30 students appear in both groups. Each before-score is naturally matched to that student's after-score.
(b) Independent
Two separate sections — different students. There is no natural pairing between an individual in one section and an individual in the other.
(c) Paired
Each patient contributes two measurements. The before and after readings are linked within the same person.
(d) Independent
Two separate cities — entirely different groups of respondents with no individual-level matching.

The key question is: can you form natural pairs, one observation from each group, that share something in common (same person, same subject, same location)? If yes, the data are paired. If the two groups consist of entirely different individuals with no link between them, the data are independent.

Problem 2 — Two-Sample \(z\)-Test for Means

A company compares the average number of units sold by two store layouts.

Layout\(n\)\(\bar{x}\)\(\sigma\)
14058.28.0
24554.67.5

Test \(H_0 : \mu_1 = \mu_2\) vs. \(H_a : \mu_1 > \mu_2\) at \(\alpha = 0.05\).

  1. Compute the standard error.
  2. Compute the test statistic \(z\).
  3. Find the \(p\)-value.
  4. State the decision and conclusion in context.
Show Solution
  1. \[ \text{SE} = \sqrt{\frac{\sigma_1^2}{n_1} + \frac{\sigma_2^2}{n_2}} = \sqrt{\frac{64}{40} + \frac{56.25}{45}} = \sqrt{1.6 + 1.25} = \sqrt{2.85} \approx 1.688 \]
    Answer: \(\text{SE} \approx 1.688\)
  2. \[ z = \frac{(\bar{x}_1 - \bar{x}_2) - 0}{\text{SE}} = \frac{58.2 - 54.6}{1.688} = \frac{3.6}{1.688} \approx 2.13 \]
    Answer: \(z \approx 2.13\)
  3. This is a right-tailed test, so the \(p\)-value is \(1 - P(Z \le 2.13)\). \[ p\text{-value} = 1 - 0.9834 = 0.0166 \]
    Answer: \(p\text{-value} \approx 0.0166\)
  4. Since \(p\text{-value} = 0.0166 < \alpha = 0.05\):
    Decision: Reject \(H_0\).
    Conclusion: There is sufficient evidence at \(\alpha = 0.05\) to conclude that Layout 1 produces a higher mean number of units sold than Layout 2.

Problem 3 — Confidence Interval for \(\mu_1 - \mu_2\)

Using the same data from Problem 2, construct a 95% confidence interval for \(\mu_1 - \mu_2\).

  1. Compute the margin of error.
  2. Write the interval.
  3. Does the interval contain 0? What does that tell you?
Show Solution
  1. For 95% confidence, \(z^* = 1.96\). Using \(\text{SE} \approx 1.688\) from Problem 2: \[ E = z^* \times \text{SE} = 1.96 \times 1.688 \approx 3.308 \]
    Answer: \(E \approx 3.31\)
  2. \[ (\bar{x}_1 - \bar{x}_2) \pm E = 3.6 \pm 3.31 \]
    Answer: \((0.29,\ 6.91)\)
  3. Answer: The interval \((0.29,\ 6.91)\) does not contain 0. Since the entire interval lies above 0, the data are consistent with \(\mu_1 > \mu_2\) at the 95% confidence level — Layout 1 outsells Layout 2 by somewhere between 0.29 and 6.91 units on average.

    An interval for \(\mu_1 - \mu_2\) that does not contain 0 corresponds exactly to rejecting \(H_0 : \mu_1 = \mu_2\) at the matching significance level. Here both the test (Problem 2) and the interval (Problem 3) agree: the layouts differ.

Problem 4 — Two-Sample \(z\)-Test for Proportions

A school compares club participation in two grades.

GradeParticipants\(n\)
1184150
1263140

Test \(H_0 : p_1 = p_2\) vs. \(H_a : p_1 \ne p_2\) at \(\alpha = 0.05\).

  1. Compute \(\hat{p}_1\) and \(\hat{p}_2\).
  2. Compute the pooled proportion \(\hat{p}_{\text{pool}}\).
  3. Compute the test statistic \(z\).
  4. Find the \(p\)-value and state the conclusion.
Show Solution
  1. \[ \hat{p}_1 = \frac{84}{150} = 0.56 \qquad \hat{p}_2 = \frac{63}{140} = 0.45 \]
    Answer: \(\hat{p}_1 = 0.56\), \(\quad \hat{p}_2 = 0.45\)
  2. \[ \hat{p}_{\text{pool}} = \frac{84 + 63}{150 + 140} = \frac{147}{290} \approx 0.5069 \]
    Answer: \(\hat{p}_{\text{pool}} \approx 0.5069\)

    The pooled proportion combines both samples into one overall estimate of the common proportion assumed under \(H_0\). We use it — not the individual \(\hat{p}\) values — in the denominator of the test statistic.

  3. \[ \text{SE} = \sqrt{\hat{p}_{\text{pool}}(1-\hat{p}_{\text{pool}})\!\left(\frac{1}{n_1}+\frac{1}{n_2}\right)} = \sqrt{0.5069 \times 0.4931 \times \left(\frac{1}{150}+\frac{1}{140}\right)} \] \[ = \sqrt{0.2500 \times 0.01381} = \sqrt{0.003452} \approx 0.05876 \] \[ z = \frac{\hat{p}_1 - \hat{p}_2}{\text{SE}} = \frac{0.56 - 0.45}{0.05876} = \frac{0.11}{0.05876} \approx 1.87 \]
    Answer: \(z \approx 1.87\)
  4. This is a two-tailed test, so the \(p\)-value is \(2(1 - P(Z \le |1.87|))\). \[ p\text{-value} = 2(1 - P(Z \le 1.87)) = 2(1 - 0.9693) = 2(0.0307) = 0.0614 \] Since \(p\text{-value} = 0.0614 > \alpha = 0.05\):
    Decision: Fail to reject \(H_0\).
    Conclusion: There is not sufficient evidence at \(\alpha = 0.05\) to conclude that club participation rates differ between Grade 11 and Grade 12.

    The result is close: \(p = 0.0614\) is just above the threshold of 0.05. This is sometimes called a "borderline" result. We do not reject \(H_0\), but a larger sample might reveal a real difference.

What to Watch For

paired vs. independent standard error formula pooled proportion CI contains 0 two-tailed p-value conclusion in context
  • Paired data require a different procedure (differences within pairs) — not covered on this prequiz but important to recognize.
  • For two-sample means, use each group's own \(\sigma\) in the SE formula. For two-sample proportions, use the pooled proportion for the test statistic.
  • A 95% CI for \(\mu_1 - \mu_2\) that does not contain 0 corresponds to rejecting \(H_0 : \mu_1 = \mu_2\) at \(\alpha = 0.05\) — the test and interval are two sides of the same inference.
  • For a two-tailed test, double the one-tail area: \(p\text{-value} = 2(1 - P(Z \le |z|))\).
  • Never say "accept \(H_0\)" — failing to reject is not the same as proving equality.
