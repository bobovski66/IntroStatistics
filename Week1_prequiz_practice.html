Reference Formulas

\[ \bar{x} = \frac{\sum x_i}{n} \qquad s = \sqrt{\frac{\sum (x_i - \bar{x})^2}{n-1}} \qquad \text{relative frequency} = \frac{\text{frequency}}{n} \]

Problem 1. Population, Sample, and Notation

A county health department wants to estimate the average number of hours of sleep adults in the county get per night. They recruit 120 randomly selected adults and record how many hours each person slept last night.

  1. Identify the population in this study.
  2. Identify the sample.
  3. The department finds the average sleep time for the 120 participants is \(6.8\) hours. Which symbol represents this value — \(\mu\) or \(\bar{x}\)? Explain why.
  4. The true average sleep time for all adults in the county is unknown. Which Greek letter would we use to represent this unknown value?
Show full solution
  1. Population: all adults in the county.
  2. Sample: the 120 randomly selected adults who were recruited for the study.
  3. The value \(6.8\) hours is the average computed from the sample, so the correct symbol is \(\bar{x}\), not \(\mu\). The symbol \(\bar{x}\) is used for a sample mean.
  4. The unknown true average for the entire population is represented by the Greek letter \(\mu\).

Key idea: \(\bar{x}\) describes a sample, while \(\mu\) describes a population.

Problem 3. Mean and Standard Deviation

Five friends compare the number of text messages they sent yesterday:

\(12, 25, 8, 31, 14\)

  1. Compute the sample mean \(\bar{x}\).
  2. Complete the deviation table, then compute the sample standard deviation \(s\). Round to two decimal places.
  3. A sixth friend joins and says they sent \(500\) text messages. Without computing, does adding this person increase, decrease, or not change \(\bar{x}\)? Do you expect \(s\) to increase or decrease? Briefly explain.
Show full solution
  1. First find the sum: \[ 12+25+8+31+14=90 \] Since there are \(n=5\) values, \[ \bar{x}=\frac{90}{5}=18 \]
  2. Compute deviations from the mean \(\bar{x}=18\):
    \(x_i\) \(x_i-\bar{x}\) \((x_i-\bar{x})^2\)
    12\(12-18=-6\)\(36\)
    25\(25-18=7\)\(49\)
    8\(8-18=-10\)\(100\)
    31\(31-18=13\)\(169\)
    14\(14-18=-4\)\(16\)
    Sum \(370\)
    Now use the sample standard deviation formula: \[ s=\sqrt{\frac{\sum (x_i-\bar{x})^2}{n-1}} =\sqrt{\frac{370}{4}} =\sqrt{92.5} \approx 9.62 \] So the sample standard deviation is \[ \boxed{s\approx 9.62} \]
  3. Adding \(500\) will increase the mean \(\bar{x}\), because \(500\) is much larger than all the original values. We also expect the standard deviation \(s\) to increase, because the new value is far from the rest of the data, so the data become much more spread out.

Notation Reminder

  • \(\bar{x}\) = sample mean
  • \(\mu\) = population mean
  • \(s\) = sample standard deviation
  • \(x_i\) = the \(i\)-th data value in a sample
  • \(n\) = sample size
