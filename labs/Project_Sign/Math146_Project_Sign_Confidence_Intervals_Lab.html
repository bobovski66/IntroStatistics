Before you begin

The Project Sign table contains 108 report entries whose number of objects was recorded as one exact whole number. For this activity, treat those 108 values as an empirical population. A fixed sample of 36 values was drawn from that population with replacement.

Interpret carefully: This setup lets you practice confidence intervals and check whether they capture a known empirical value. It does not make the archival reports a random sample of all aerial events.

Objectives

  • Construct a confidence interval for a population mean when \(\sigma\) is known.
  • Explain how confidence level changes interval width.
  • Find a required sample size for a target margin of error.
  • Construct and interpret a confidence interval for a population proportion.
  • Explain the repeated-sampling meaning of 95% confidence.

Reference formulas

\[ \bar{x}\pm z^*\frac{\sigma}{\sqrt n} \qquad\qquad \hat p\pm z^*\sqrt{\frac{\hat p(1-\hat p)}{n}} \] \[ n\ge\left(\frac{z^*\sigma}{\operatorname{ME}}\right)^2 \]

Critical values: \(90\%:z^*=1.645\), \(95\%:z^*=1.960\), \(99\%:z^*=2.576\).

Part 1. A 95% confidence interval for a mean

Let \(X\) be the number of objects reported in one selected entry. Use the sample below.

For this empirical population, use the known standard deviation \[\sigma=3.028\text{ objects}.\]

\[ \bar x=1.944, \qquad \operatorname{SE}=\frac{3.028}{\sqrt{36}}=0.505, \qquad \operatorname{ME}=1.96(0.505)=0.989. \] The 95% confidence interval is \((0.955,\,2.934)\).
The known empirical-population mean is \(\mu=2.213\). Determine whether your interval contains this value.

The 90% interval is \((1.114,2.775)\), and the 99% interval is \((0.645,3.244)\). The 99% interval is wider because it uses a larger critical value.

Part 3. Required sample size

Suppose you want a 95% confidence interval for the mean number reported with margin of error no larger than \(0.50\) object. Continue to use \(\sigma=3.028\).

\[n\ge\left(\frac{1.96(3.028)}{0.50}\right)^2\]

\[n\ge 140.873,\] so the required sample size is \(n=141\). Round up so that the margin of error does not exceed the target.

Part 4. A 95% confidence interval for a proportion

Define a success as a sampled entry reporting exactly one object. Use the same sample of 36 values.

Check the success–failure condition

The normal confidence interval is appropriate when \(n\hat p\ge10\) and \(n(1-\hat p)\ge10\).

\[ \hat p=\frac{25}{36}=0.6944, \qquad \operatorname{SE}=\sqrt{\frac{\hat p(1-\hat p)}{36}}=0.0768, \] \[ \operatorname{ME}=1.96(0.0768)=0.1505, \qquad 95\%\text{ CI}=(0.5440,0.8449). \]
The known empirical proportion is \[p=\frac{76}{108}=0.7037.\] Does your confidence interval contain it?
