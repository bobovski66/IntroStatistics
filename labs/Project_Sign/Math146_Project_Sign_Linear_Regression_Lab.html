Before you begin

The analysis uses the 102 Project Sign entries that have both an exact calendar date and an exact whole-number object count.

  • \(x=\) days since January 1, 1947
  • \(y=\) number of objects reported
Interpret carefully: A regression line describes association in these recorded entries. It does not show that time caused the reports to change.

Objectives

  • Describe direction, form, strength, and outliers in a scatterplot.
  • Compute a least-squares regression line from summary statistics.
  • Interpret the slope, correlation, and coefficient of determination.
  • Make a prediction and calculate a residual.
  • Recognize weak fit, influential observations, and extrapolation.

Reference formulas

\[ b=r\frac{s_y}{s_x},\qquad a=\bar y-b\bar x,\qquad \hat y=a+bx \] \[ e=y-\hat y,\qquad r^2=\text{proportion of variability in }y\text{ explained by the linear model} \]

Part 3. Interpret the model

The slope is \(-0.00973\) object per day, or about \(-0.292\) object per 30 days. The model explains \(6.67\%\) of the variation in recorded object count.
