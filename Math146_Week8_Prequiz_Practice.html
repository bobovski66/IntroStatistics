Reference: Chi-Squared Tests

Test statistic: \[\chi^2=\sum \frac{(O-E)^2}{E}\]

Goodness-of-fit expected counts: \(E_i=n p_i\)

Independence expected counts: \[E=\frac{(\text{row total})(\text{column total})}{\text{grand total}}\]

Degrees of freedom: goodness-of-fit \(df=k-1\); independence \(df=(r-1)(c-1)\)

For chi-squared tests, the p-value is always the area to the right of the test statistic.

Problem 1 — Goodness-of-Fit: Campus Event Preferences

A student activities survey asks 160 students which evening event they would most like to attend. The null hypothesis says the four choices are equally popular.

  3. Compute each contribution \((O-E)^2/E\).
  4. Find \(\chi^2\) and \(df\).

Hypotheses. \(H_0\): the four event choices are equally popular. \(H_a\): the four event choices are not equally popular.

Since \(n=160\) and \(k=4\), the expected count is \(E=160/4=40\) for each category.

Contribution4.902.500.908.10

\(\chi^2=4.90+2.50+0.90+8.10=16.40\). Also, \(df=k-1=4-1=3\).

For \(df=3\), \(16.40>16.266\), so \(p<0.001\). Since \(p<0.05\), reject \(H_0\).

Conclusion: There is sufficient evidence to conclude that the four event choices are not equally popular.

Problem 3 — Test of Independence: Study Location and Quiz Result

A class survey records preferred study location and whether the student passed the next quiz.

Library361450

Cafe161430
Column total7644120
  1. State \(H_0\) and \(H_a\).
  2. Compute the expected counts for Library/Passed and Library/Did not pass.
  3. Find \(df\).
  4. Suppose the full calculation gives \(\chi^2=3.35\). Use the table to make a decision at \(\alpha=0.05\).
  5. Write a conclusion using the word associated.
Show Solution

\(H_0\): study location and quiz result are independent. \(H_a\): study location and quiz result are associated.

Library/Passed: \(E=(50)(76)/120=31.67\). Library/Did not pass: \(E=(50)(44)/120=18.33\).

The table is \(3\times2\), so \(df=(3-1)(2-1)=2\).

For \(df=2\), \(3.35<4.605\), so \(p>0.10\). We fail to reject \(H_0\) at \(\alpha=0.05\).

Conclusion: There is insufficient evidence to conclude that preferred study location and quiz result are associated.

Problem 4 — Reading the Chi-Squared Table

For each statistic, bracket the p-value as best you can and decide whether to reject at \(\alpha=0.05\).

  1. \(\chi^2=8.2\), \(df=3\)
  2. \(\chi^2=14.0\), \(df=4\)
  3. \(\chi^2=5.4\), \(df=2\)
Show Solution

(a) For \(df=3\), \(7.815<8.2<11.345\), so \(0.01<p<0.05\). Reject at \(\alpha=0.05\).

(b) For \(df=4\), \(13.277<14.0<18.467\), so \(0.001<p<0.01\). Reject at \(\alpha=0.05\).

(c) For \(df=2\), \(4.605<5.4<5.991\), so \(0.05<p<0.10\). Fail to reject at \(\alpha=0.05\).
