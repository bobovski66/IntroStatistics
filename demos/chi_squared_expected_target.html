Chi-Squared Geometry in a Triangle

This demo focuses on three categories, where fixed-total count data can be drawn inside a triangle. The orange point is the expected count vector from a chosen null model. The blue point is a random observed count vector. Use the sliders to move the observed data toward the expected model and watch the chi-squared distance shrink.

Goal: The orange point is the expected model. Move the blue observed point until it lands on the orange expected point. The chi-squared distance gets smaller as you match the model.

Weighted distance calculation

\[ \chi^2=\sum \frac{(O-E)^2}{E} =\left\|\left(\frac{O_1-E_1}{\sqrt{E_1}},\frac{O_2-E_2}{\sqrt{E_2}},\frac{O_3-E_3}{\sqrt{E_3}}\right)\right\|^2 \]

Standardized vector components

These are the coordinates of \(z=(O-E)/\sqrt{E}\). The chi-squared statistic is the sum of their squares.

Student-facing point: for three categories, the possible count proportions form a triangle. The chi-squared test measures how far the observed point is from the expected-model point, after rescaling each coordinate by its expected count size.
