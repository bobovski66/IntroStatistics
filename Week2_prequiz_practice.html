Reference Formulas

\[ P(\text{Geometric: } X=k)=(1-p)^{k-1}p \qquad \mu_{\text{geo}}=\frac{1}{p} \]

\[ P(\text{Binomial: } X=k)=\binom{n}{k}p^k(1-p)^{n-k} \qquad \mu_{\text{bin}}=np \qquad \sigma_{\text{bin}}=\sqrt{np(1-p)} \]

\[ \binom{n}{k}=\frac{n!}{k!(n-k)!} \qquad E(X)=\sum x\cdot P(X=x) \]

Problem 1. Probability Distribution

A spinner is divided into four sectors: the spinner lands on \(1\) with probability \(0.10\), on \(2\) with probability \(0.30\), on \(3\) with probability \(0.40\), and on \(4\) with probability \(0.20\). Let \(X\) be the number the spinner lands on.

  1. Complete the probability distribution table.
  2. Verify that the probabilities sum to \(1\).
  3. Sketch the probability histogram. Label both axes, and make the bar heights match the probabilities.
  4. Compute \(\mu = E(X)\).
\(x\) 1 2 3 4
\(P(X=x)\) 0.10   0.40  
Show full solution
  1. The completed table is:
    \(x\) 1 2 3 4
    \(P(X=x)\) 0.10 0.30 0.40 0.20
  2. Add the probabilities: \[ 0.10+0.30+0.40+0.20=1.00 \] So this is a valid probability distribution.
  3. The histogram has four bars:
    \(x=1\)
    height \(0.10\)
    \(x=2\)
    height \(0.30\)
    \(x=3\)
    height \(0.40\)
    \(x=4\)
    height \(0.20\)
    Horizontal axis: outcome \(x\)
    Vertical axis: \(P(X=x)\)
  4. Compute the expected value: \[ E(X)=1(0.10)+2(0.30)+3(0.40)+4(0.20) \] \[ =0.10+0.60+1.20+0.80=2.70 \] So \[ \boxed{\mu=E(X)=2.7} \]

The expected value is the long-run average outcome of the spinner.

Problem 2. Geometric Distribution

A student randomly guesses on each question of a multiple-choice exam where each question has \(5\) choices and exactly one correct answer. Let \(X\) be the question number on which the student first guesses correctly.

  1. What is the success probability \(p\) on each question?
  2. Compute \(P(X=1)\), \(P(X=2)\), and \(P(X=4)\).
  3. Compute \(\mu\). Interpret this number in a sentence.
  4. Sketch the first 5 bars of the probability histogram. Is it skewed left, skewed right, or symmetric?
Show full solution
  1. There is \(1\) correct answer out of \(5\) choices, so \[ p=\frac{1}{5}=0.20 \]
  2. Use the geometric formula \[ P(X=k)=(1-p)^{k-1}p \] with \(p=0.20\), so \(1-p=0.80\).

    For \(X=1\): \[ P(X=1)=(0.80)^0(0.20)=1(0.20)=0.20 \]

    For \(X=2\): \[ P(X=2)=(0.80)^1(0.20)=0.16 \]

    For \(X=4\): \[ P(X=4)=(0.80)^3(0.20)=0.512(0.20)=0.1024 \]

  3. For a geometric distribution, \[ \mu=\frac{1}{p}=\frac{1}{0.20}=5 \] So \[ \boxed{\mu=5} \] Interpretation: on average, the student will get the first correct answer on about the \(5\)th question.
  4. The first five bar heights are:
    \(k=1\)
    \(0.2000\)
    \(k=2\)
    \(0.1600\)
    \(k=3\)
    \(0.1280\)
    \(k=4\)
    \(0.1024\)
    \(k=5\)
    \(0.08192\)
    Horizontal axis: question number \(k\)
    Vertical axis: \(P(X=k)\)
    Since the bars start high and then decrease as \(k\) increases, the histogram is skewed right.

Problem 3. Binomial Distribution

A pharmaceutical company tests a new drug. In clinical trials, the drug is effective for \(70\%\) of patients. A doctor prescribes it to \(6\) new patients. Let \(X\) be the number of patients for whom the drug is effective.

  1. Identify \(n\) and \(p\). Explain why this is a binomial setting.
  2. Compute \(P(X=6)\).
  3. Compute \(P(X=4)\). Show the combination calculation.
  4. Compute \(\mu\) and \(\sigma\). Interpret \(\mu\).
  5. Without computing every bar, describe what you would expect the histogram to look like: symmetric, skewed left, or skewed right? At approximately what value of \(k\) would the tallest bar be?
Show full solution
  1. We have \[ n=6 \qquad \text{and} \qquad p=0.70 \] This is binomial because:
    • there is a fixed number of trials (\(6\) patients),
    • each trial has two outcomes (effective or not effective),
    • the probability of success stays the same (\(0.70\)),
    • and the trials are treated as independent.
  2. Use the binomial formula: \[ P(X=6)=\binom{6}{6}(0.70)^6(0.30)^0 \] Since \(\binom{6}{6}=1\) and \((0.30)^0=1\), \[ P(X=6)=(0.70)^6=0.117649 \] So \[ \boxed{P(X=6)\approx 0.1176} \]
  3. Now compute \[ P(X=4)=\binom{6}{4}(0.70)^4(0.30)^2 \] First, \[ \binom{6}{4}=\frac{6!}{4!2!}=\frac{6\cdot 5}{2\cdot 1}=15 \] Then \[ P(X=4)=15(0.70)^4(0.30)^2 \] \[ =15(0.2401)(0.09)=15(0.021609)=0.324135 \] So \[ \boxed{P(X=4)\approx 0.3241} \]
  4. For a binomial distribution, \[ \mu=np=6(0.70)=4.2 \] and \[ \sigma=\sqrt{np(1-p)}=\sqrt{6(0.70)(0.30)}=\sqrt{1.26}\approx 1.1225 \] So \[ \boxed{\mu=4.2 \qquad \sigma\approx 1.12} \] Interpretation of \(\mu\): on average, about \(4.2\) out of the \(6\) patients would respond effectively to the drug.
  5. Since \(p=0.70\) is greater than \(0.50\), the distribution is not symmetric. It is skewed left, with more probability concentrated near the larger values of \(k\). The tallest bar should be around \(k=4\), which is close to the mean \(4.2\) and in fact matches the large probability we computed for \(P(X=4)\).

What to Watch For

These are the main ideas from this week’s practice:

probability distribution expected value geometric distribution binomial distribution histogram shape
  • For any probability distribution, the probabilities must add up to \(1\).
  • Expected value is a weighted average: multiply each outcome by its probability.
  • Geometric distributions model the first success.
  • Binomial distributions model the number of successes in a fixed number of trials.
