Math 146 — Week 6 Prequiz Practice

Hypothesis testing for means and proportions. Expand each solution to check your work.

Reference Formulas

Test statistic for a mean: \[ z = \frac{\bar{x} - \mu_0}{\sigma / \sqrt{n}} \]

Test statistic for a proportion: \[ z = \frac{\hat{p} - p_0}{\sqrt{p_0(1-p_0)/n}} \]

Left-tailed \(p\)-value: \(P(Z \le z)\)    Right-tailed \(p\)-value: \(1 - P(Z \le z)\)    Two-tailed \(p\)-value: \(2\bigl(1 - P(Z \le |z|)\bigr)\)

Problem 1 — Writing Hypotheses

For each scenario, state \(H_0\) and \(H_a\), and identify the test type.

  1. A cereal box is supposed to contain an average of 18 ounces. A consumer group suspects the boxes contain less.
  2. A school says the average commute time for its students is 24 minutes. A researcher wants to know whether that average has changed.
  3. A website claims that 70% of users finish registration. A manager believes the true proportion is higher.
Show Solution
  1. The claim to be challenged is that the mean is 18 oz. "Suspects less" gives the direction of \(H_a\).
    \(H_0\):\(\mu = 18\) oz
    \(H_a\):\(\mu < 18\) oz
    Test type:Left-tailed
  2. "Changed" does not give a direction, so the alternative is two-sided.
    \(H_0\):\(\mu = 24\) min
    \(H_a\):\(\mu \ne 24\) min
    Test type:Two-tailed
  3. The claim is \(p = 0.70\). "Believes the true proportion is higher" gives the direction of \(H_a\).
    \(H_0\):\(p = 0.70\)
    \(H_a\):\(p > 0.70\)
    Test type:Right-tailed

\(H_0\) always contains equality. The sign in \(H_a\) is determined by the research question: one directional word ("less," "more," "higher," "lower") gives a one-tailed test; "changed," "different," or "not equal" gives a two-tailed test.

Problem 2 — One-Sample \(z\)-Test for a Mean

A machine is supposed to fill bottles with a mean of 20.0 ounces. A technician samples 49 bottles and finds \(\bar{x} = 19.7\) ounces. Assume \(\sigma = 0.8\) ounces and test

\[ H_0 : \mu = 20.0 \quad \text{vs.} \quad H_a : \mu < 20.0 \]

at \(\alpha = 0.05\).

  1. Compute the test statistic \(z\).
  2. Find the \(p\)-value.
  3. State the decision.
  4. Write the conclusion in context.
Show Solution
  1. \[ z = \frac{\bar{x} - \mu_0}{\sigma/\sqrt{n}} = \frac{19.7 - 20.0}{0.8/\sqrt{49}} = \frac{-0.3}{0.8/7} = \frac{-0.3}{0.1143} \approx -2.63 \]
    Answer: \(z \approx -2.63\)
  2. This is a left-tailed test, so the \(p\)-value is \(P(Z \le -2.63)\). \[ p\text{-value} = P(Z \le -2.63) \approx 0.0043 \]
    Answer: \(p\text{-value} \approx 0.0043\)
  3. Since \(p\text{-value} = 0.0043 < \alpha = 0.05\):
    Decision: Reject \(H_0\).
  4. Conclusion: There is sufficient evidence at the \(\alpha = 0.05\) level to conclude that the machine is underfilling bottles — the true mean fill amount appears to be less than 20.0 ounces.

    The test statistic \(z = -2.63\) is well into the left tail. The probability of seeing a sample mean this far below 20.0 by chance alone, when \(H_0\) is true, is less than 0.5%.

Problem 3 — Interpreting a \(p\)-Value

A test gives \(z = 1.84\).

  1. Find the \(p\)-value for a right-tailed test.
  2. Find the \(p\)-value for a two-tailed test.
  3. If \(\alpha = 0.05\), what is the decision for each test?
Show Solution
  1. Right-tailed \(p\)-value \(= 1 - P(Z \le 1.84)\). From the \(z\)-table, \(P(Z \le 1.84) = 0.9671\). \[ p\text{-value} = 1 - 0.9671 = 0.0329 \]
    Answer: \(p\text{-value} = 0.0329\)
  2. Two-tailed \(p\)-value \(= 2(1 - P(Z \le |z|)) = 2(1 - 0.9671)\): \[ p\text{-value} = 2 \times 0.0329 = 0.0658 \]
    Answer: \(p\text{-value} = 0.0658\)
  3. Test type\(p\)-value\(\alpha\)Decision
    Right-tailed 0.0329 0.05 \(0.0329 < 0.05\)  →  Reject \(H_0\)
    Two-tailed 0.0658 0.05 \(0.0658 > 0.05\)  →  Fail to reject \(H_0\)

    The same test statistic leads to opposite decisions depending on the test type. This illustrates why the direction of the alternative hypothesis must be specified before seeing the data. Choosing the test type after looking at the data would inflate the Type I error rate.

Problem 4 — One-Sample \(z\)-Test for a Proportion

A company claims that 65% of customers renew their subscription. In a sample of 200 customers, 118 renew. Test

\[ H_0 : p = 0.65 \quad \text{vs.} \quad H_a : p < 0.65 \]

at \(\alpha = 0.05\).

  1. Compute \(\hat{p}\).
  2. Compute the test statistic \(z\).
  3. Find the \(p\)-value.
  4. State the decision and conclusion in context.
Show Solution
  1. \[ \hat{p} = \frac{118}{200} = 0.59 \]
    Answer: \(\hat{p} = 0.59\)
  2. The test statistic uses \(p_0 = 0.65\) (the null value) in the denominator, not \(\hat{p}\): \[ z = \frac{\hat{p} - p_0}{\sqrt{p_0(1-p_0)/n}} = \frac{0.59 - 0.65}{\sqrt{0.65 \times 0.35 / 200}} = \frac{-0.06}{\sqrt{0.001138}} = \frac{-0.06}{0.0337} \approx -1.78 \]
    Answer: \(z \approx -1.78\)
  3. This is a left-tailed test, so the \(p\)-value is \(P(Z \le -1.78)\). \[ p\text{-value} = P(Z \le -1.78) \approx 0.0375 \]
    Answer: \(p\text{-value} \approx 0.0375\)
  4. Since \(p\text{-value} = 0.0375 < \alpha = 0.05\):
    Decision: Reject \(H_0\).
    Conclusion: There is sufficient evidence at the \(\alpha = 0.05\) level to conclude that the true proportion of customers who renew their subscription is less than 65%.

    Note that the denominator uses \(p_0 = 0.65\), not \(\hat{p} = 0.59\). Under the null hypothesis we assume \(p = p_0\) is true, so that is the value we use to compute the standard error for the test.

What to Watch For

null vs. alternative test direction test statistic \(p\)-value reject vs. fail to reject \(p_0\) in denominator conclusion in context
  • \(H_0\) always contains equality (\(=\)). \(H_a\) contains \(<\), \(>\), or \(\ne\) — never \(=\).
  • The test direction (left, right, two-tailed) is determined by \(H_a\), not by where the test statistic lands.
  • For a proportion test, use \(p_0\) — the null value — in the denominator of \(z\), not \(\hat{p}\).
  • We reject \(H_0\) when \(p\text{-value} < \alpha\), and fail to reject when \(p\text{-value} \ge \alpha\). We never "accept \(H_0\)."
  • A conclusion must be written in context — name the variable and population, not just "reject \(H_0\)."
